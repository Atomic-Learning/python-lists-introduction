Python lists are a way to create and work with ordered collections of items. This page covers Python's specific syntax for creating, accessing, and manipulating lists.

Creating Lists

Lists are created using square brackets [] with items separated by commas:

fruits = ["apple", "banana", "orange"] numbers = [1, 2, 3, 4, 5] mixed = ["a", 1, True, 2.5] print(fruits) print(numbers) print(mixed)

Empty Lists

You can create an empty list with just empty brackets:

empty_list = [] print(empty_list)

Lists Can Contain Different Types

A list can contain items of any type, and even mix types:

mixed_list = ["hello", 42, 3.14, True, [1, 2, 3]] print(mixed_list)
